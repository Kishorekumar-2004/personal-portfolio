# 🌐 Kishore Kumar K — Personal Portfolio

A colorful, creative, and fully responsive personal portfolio website built from scratch using pure HTML, CSS, and JavaScript.

---

## 🚀 Live Preview

> Host on GitHub Pages, Netlify, or Vercel — see deployment steps below.

---

## ✨ Features

- **Sticky Navigation** — smooth scroll links to all sections
- **Animated Hero** — floating background blobs + fade-in entrance animations
- **About Me** — bio, skill tags, and quick stats card
- **Skills** — categorized cards with colorful pill badges
- **Projects** — hover-animated project cards with tech tags
- **Education** — vertical timeline with color-coded milestones
- **Contact Section** — gradient footer with direct links
- **Scroll Reveal Animations** — sections animate in as you scroll
- **Fully Responsive** — works on mobile, tablet, and desktop

## 🌍 Deployment

### GitHub Pages (Free & Easy)

1. Push your code to a GitHub repo named `portfolio`
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save** — your site will be live at:
   `https://Kishorekumar-2004.github.io/portfolio`

### Netlify (Drag & Drop)

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your project folder
3. Done — live in seconds!
